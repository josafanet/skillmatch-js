# SkillMatch JS

## Sobre o projeto

O SkillMatch JS sera um simulador de compatibilidade entre uma pessoa candidata e vagas ficticias de Front-End Junior.

O projeto esta sendo desenvolvido em JavaScript puro como atividade avaliativa do Modulo 01 do curso de Programacao Front-End.

## Objetivo inicial

Construir um sistema capaz de:

- armazenar os dados de um candidato;
- armazenar vagas ficticias;
- comparar habilidades com requisitos;
- indicar compatibilidade e pontos de estudo.
